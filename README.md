# cligen.vim

## Description

This is a Vim plugin that provides [CLIgen][c] file detection, syntax highlighting, formatting.
It requires Vim 7.4 or higher for full functionality. (Tested on it.)

## Installations

Currently just copy:
cp -r ./{ftdetect,ftplugin,indent,syntax} ~/.vim/
